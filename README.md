# 🔗 URL Shortener (Node.js + Express + MongoDB)

A simple URL Shortener web application built using:

- Node.js
- Express.js
- MongoDB
- EJS
- Mongoose

---

## 🚀 Features

- Generate Short URL
- Redirect using Short ID
- Track Analytics (Click Count)
- Copy Short URL

---

## 🛠 Installation

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file and add:

PORT=8001
MONGO_URI=your_mongodb_connection_string

4. Run `npm start`
5. Open `http://localhost:8001`
